# Human Motion Analysis

Expo React Native prototype for the consumer-first, iPhone-biased version of the human motion analysis idea.

## Current Scope

- iPhone-first import flow using Photos
- Portrait app shell with landscape clip expectation
- Mobile replay surface with trace / compare / raw modes
- Mock analysis states and coaching metrics wired into the UI
- Live bridge back to the vault plan through `.noggin/`

## Start Here

1. Read `.noggin/PROJECT.md`
2. Read `.noggin/TODOS.md`
3. Read `.noggin/design.md`

## Commands

```bash
npm start
npm run web
npm run ios
```

`npm run ios` requires macOS for the simulator. On Linux, use Expo Go or `npm run web` for layout validation.
